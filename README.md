# Markov Chains Discord Bot

See [here](https://github.com/krypt0nn/markov-chains) for markov chains implementation used by this bot.

## Supported commands
- /params
    - set
    - reset
    - show
- /model
    - query
    - list
    - info
    - load
    - build
    - fromscratch

## Usage
```
DISCORD_TOKEN=your_token ./markov-chains-bot
```
